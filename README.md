# Visualize
Visualize is a Hybrid Geometric Music Visualizer that works in 3 different modes (Mic Input, Free Mode, MP3 Upload), It allos for user selected Visuali Colors and Background Color.

<img width="1512" height="867" alt="Screenshot 2026-06-14 at 10 56 34 AM" src="https://github.com/user-attachments/assets/ae0ec97f-069f-4296-b90b-eb92b7886939" />

Mic Input
Allow for the program to interept mic audio and sync the visualizer to it

Free Mode
Is a Free form mode that randomly gernerates visualizers with no music sync

MP3 Upload
This mode allows for a user to choose and upload MP3's to be synced to the music visualizer
