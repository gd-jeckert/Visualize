# Visualize
Visualize is a Hybrid Geometric Music Visualizer that works in 3 different modes (Mic Input, Free Mode, MP3 Upload), It allows for user selected Visualizer Colors and Background Color.

This has morphed a little bit as I have added MP3 player playlist support for the release of Version 1.0

Mic Input
Allow for the program to interept mic audio and sync the visualizer to it

Free Mode
Is a Free form mode that randomly gernerates visualizers with no music sync

MP3 Upload
This mode allows for a user to choose and upload MP3's to be synced to the music visualizer
